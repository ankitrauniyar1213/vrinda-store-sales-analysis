# vrinda-store-sales-analysis
Sales Analysis Dashboard built using Microsoft Excel. This project analyzes sales performance, customer demographics, sales channels, and order trends to generate business insights.

Vrinda Store Sales Analysis Dashboard

Project Overview

This project analyzes Vrinda Store's 2022 sales data using Microsoft Excel.

Tools Used

2.Pivot Tables
3.Pivot Charts
4.Slicers
5.Data Visualization

Key Insights
